## Install Package
```
wget -O entangle.sh https://raw.githubusercontent.com/sipalingnode/entangle/main/entangle.sh; chmod +x entangle.sh; ./entangle.sh
```
## Konfigurasi Node
```
cd entangle-blockchain
```
```
make install
```
## Create Wallet
```
entangled keys add wallet
```
